## 🥑 Avocado — A Modern Theme for Flarum

One of the most eye-catching extensions included in Flarum-In-A-Box is [**Avocado**](https://discuss.flarum.org/d/38940-avocado-theme) by [ramon](https://discuss.flarum.org/u/ramon) — a polished, premium-feeling theme that completely transforms the look and feel of your forum.

### How to Enable It

1. Log in as **admin** (password: `password`)
2. Go to **Admin Panel → Extensions**
3. Find **Avocado** and click to enable it
4. Refresh the page — enjoy the new look!

### What It Changes

- 🖼️ **Hero banner** — Add a custom banner image to the top of your forum
- 🏷️ **Tag-based styling** — Discussions are styled with their tag colors
- 📌 **Sticky avatars** — Avatars stay visible while scrolling through discussions
- 🔄 **Smart share button** — Native Web Share API on mobile, clipboard fallback on desktop
- 🎨 **Vibrant customization** — Tweak colors, banners, and icons from the admin panel
- ✨ **Smooth animations** — Preloader, focus effects, and crisp typography
- 📱 **Mobile-first** — Optimized navigation and modals for great mobile UX

### Why Try It?

Avocado is a great example of how Flarum themes can dramatically change the user experience with just one click. It is perfect for:

- Seeing how theme extensions work in Flarum
- Getting inspiration for your own forum styling
- Showing stakeholders different visual options

Give it a try and see the difference! You can always disable it to go back to the default look.
