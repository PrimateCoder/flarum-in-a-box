## Extensions Guide

This instance comes with **~50 extensions** installed. Visit **Admin Panel → Extensions** to see and configure them all.

### Enabled by Default

Core: Tags, Likes, Mentions, Lock, Sticky, Suspend, Markdown, BBCode, Emoji, Flags, Nicknames, Subscriptions, Approval, Statistics

Community favorites: **Flamoji** (emoji picker), **Upload**, **Polls**, **Best Answer**, **Byobu** (private discussions), **Drafts**, **Reactions**, **Rich Text**, **Gamification** (voting & rankings), **Profile Cover**, **Mobile Tab**, **Categories**, **Stickiest**, **Diff** (edit history), **Synopsis**, **Discussion Views**, **Impersonate**, **Split/Merge**, **Topic Rating**, **Post Search**, **Forum Widgets**, **Markdown Tables**, **Inline Audio**, and more.

### Installed but Not Enabled

Some extensions are installed but disabled by default — try them out!

- 🥑 **Avocado** — A modern, polished theme with hero banner, tag styling, and rich customization (see the dedicated post about it!)
- 🎨 **Colored** — Colorful usernames by group
- 🦶 **Modern Footer** — Responsive footer
- 🔤 **Font Sizer** — Adjustable font sizes
- 📄 **FoF Pages** — Custom static pages
- 🖼️ **FoF Discussion Thumbnail** — Thumbnails in discussion list
- 🤝 **FoF Terms** — Terms of service acceptance
- 📣 **FoF Share Social** — Social media sharing
- 🛡️ **FoF Anti Spam** — Spam prevention

Enable any of them from the Admin Panel → Extensions.

### Installing More Extensions

The **Extension Manager** is enabled — search for and install additional extensions directly from the Admin Panel, no command line needed.
