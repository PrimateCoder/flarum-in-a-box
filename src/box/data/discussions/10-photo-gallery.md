### Try it yourself

1. Click any image above — **PhotoSwipe** opens it full-screen
2. Use **← →** to navigate, or swipe on mobile
3. Pinch or scroll to zoom in
4. Press **Esc** or swipe down to close

### How it works

**PhotoSwipe** is a passive enhancement — it binds to every `<img>` in a post automatically. No special markdown, no shortcodes. Just paste an image and it works.

Try uploading your own image with the [Upload](https://packagist.org/packages/fof/upload) extension (paperclip icon in the composer) and reply — it'll join the gallery.

*Images are random photos from [Picsum](https://picsum.photos) — each `seed` parameter pins one specific image, so the gallery looks the same across container restarts.*
